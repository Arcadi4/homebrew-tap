# Homebrew Tap

My personal Homebrew tap, mostly projects I am using or interested in.

## Usage

```sh
brew tap arcadi4/tap
```

Install a formula or cask with:

```sh
brew install <name>
```

## Formulae

| Name | Description |
| --- | --- |
| `markitdown` | Convert files to Markdown for LLMs and text analysis. |

## Casks

| Name | Description |
| --- | --- |
| `pi-mono` | AI agent toolkit with CLI, unified LLM API, TUI, and web UI libraries. |
| `sidekick` | Native macOS app for chatting with a local LLM that can use information from files, folders, and websites on your Mac. |
